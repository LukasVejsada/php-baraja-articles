Availability in `PHP 4.0`

Tells whether the filename is a symbolic link


Parameters
--------------

| Parameter | Data type | Default value | Note |
|-----|-----|-----|-----|
| `$filename` | `string` | *not* | Path to the file. |


Return values
----------------

`bool`

true if the filename exists and is a symbolic link, false
otherwise.
