Availability in `PHP 4.0`

Tells whether the filename is writable


Parameters
--------------

| Parameter | Data type | Default value | Note |
|-----|-----|-----|-----|
| `$filename` | `string` | *not* | The filename being checked. |


Return values
----------------

`bool`

true if the filename exists and is
writable.
