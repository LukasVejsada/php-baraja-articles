Push one or more elements onto the end of array


Parameters
--------------

| Parameter | Data type | Default value | Note |
|-----|-----|-----|-----|
| `$array` | `array` | *not* | The input array. |
| `$var` | `mixed` | *not* | The pushed value. |
| `$_` | `mixed` | null | |


Return values
----------------

`int`

the new number of elements in the array.
