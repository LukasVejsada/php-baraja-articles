Availability in `PHP 4.0`

Delay execution in microseconds


Parameters
--------------

| Parameter | Data type | Default value | Note |
|-----|-----|-----|-----|
| `$micro_seconds` | `int` | *not* | Halt time in micro seconds. A micro second is one millionth of a second. |


Return values
----------------

`void`
