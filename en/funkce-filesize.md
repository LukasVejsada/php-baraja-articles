Availability in `PHP 4.0`

Gets file size


Parameters
--------------

| Parameter | Data type | Default value | Note |
|-----|-----|-----|-----|
| `$filename` | `string` | *not* | Path to the file. |


Return values
----------------

`int`

the size of the file in bytes, or false (and generates an error
of level E_WARNING) in case of an error.
