Availability in `PHP 4.0`

Adds slashes to the string in appropriate places so that the content is escaped.

Parameters
--------------

| Parameter | Data Type | Default Value | Note |
|-----|-----|-----|-----|
| `$str` | `string` | *not* | String to be escaped. |


Return values
----------------

`string` - escaped string.
