Returns available space in directory


Parameters
--------------

| Parameter | Data type | Default value | Note |
|-----|-----|-----|-----|
| `$directory` | `string` | *not* | A directory of the filesystem or disk partition. |


Return values
----------------

`float`

|bool the number of available bytes as a float
or false on failure.
