Returns the total size of a directory


Parameters
--------------

| Parameter | Data type | Default value | Note |
|-----|-----|-----|-----|
| `$directory` | `string` | *not* | A directory of the filesystem or disk partition. |


Return values
----------------

`float`

|bool the total number of bytes as a float
or false on failure.
