Availability in versions: `PHP 4.0.4`

Finds whether a variable is &null;


Parameters
--------------

| Parameter | Data type | Default value | Note |
|-----|-----|-----|-----|
| `$var` | `mixed` | *not* | The variable being evaluated. |


Return values
----------------

`bool`

true if var is null, false
otherwise.
