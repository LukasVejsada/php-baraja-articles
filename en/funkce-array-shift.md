Availability in `PHP 4.0`

Shift an element off the beginning of array


Parameters
--------------

| Parameter | Data type | Default value | Note |
|-----|-----|-----|-----|
| `$array` | `array` | *not* | The input array. |


Return values
----------------

`mixed`

the shifted value, or &null; if array is
empty or is not an array.
