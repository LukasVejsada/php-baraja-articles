Sets the value of a configuration option


Parameters
--------------

| Parameter | Data type | Default value | Note |
|-----|-----|-----|-----|
| `$varname` | `string` | *not* | |
| `$newvalue` | `string` | *not* | The new value for the option. |


Return values
----------------

`string`

|bool the old value on success, false on failure.
