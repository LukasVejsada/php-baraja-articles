Availability in `PHP 4.0`

Finds whether a variable is a resource


Parameters
--------------

| Parameter | Data type | Default value | Note |
|-----|-----|-----|-----|
| `$var` | `mixed` | *not* | The variable being evaluated. |


Return values
----------------

`bool`

true if var is a resource,
false otherwise.
