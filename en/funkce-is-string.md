Availability in `PHP 4.0`

Find whether the type of a variable is string


Parameters
--------------

| Parameter | Data type | Default value | Note |
|-----|-----|-----|-----|
| `$var` | `mixed` | *not* | The variable being evaluated. |


Return values
----------------

`bool`

true if var is of type string,
false otherwise.
