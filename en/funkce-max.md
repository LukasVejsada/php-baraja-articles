Availability in `PHP 4.0`

Returns the largest element of the array.


Parameters
--------------

| Parameter | Data type | Default value | Note |
|-----|-----|-----|-----|
| `$value1` | `array|mixed` | *not* | Searched array |
| `$value2` | `mixed` | null, | |
| `$values` | `mixed` | *not* | |


Return values
----------------

`mixed`

Returns the largest element of the array.

```php
max([1, 5, 3, 6, 2, 4]); // returns 6
```
