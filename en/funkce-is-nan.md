Availability in versions: `PHP 4.2.0`

Finds whether a value is not a number


Parameters
--------------

| Parameter | Data type | Default value | Note |
|-----|-----|-----|-----|
| `$val` | `float` | *not* | The value to check |


Return values
----------------

`bool`

true if val is 'not a number',
else false.
