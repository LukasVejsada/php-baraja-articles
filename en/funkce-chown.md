Availability in `PHP 4.0`

Changes file owner


Parameters
--------------

| Parameter | Data type | Default value | Note |
|-----|-----|-----|-----|
| `$filename` | `string` | *not* | Path to the file. |
| `$user` | `mixed` | *not* | A user name or number. |


Return values
----------------

`bool`

Returns `true` on success, otherwise `false` on failure.
