Returns the floating point remainder (modulo) of the division


Parameters
--------------

| Parameter | Data type | Default value | Note |
|-----|-----|-----|-----|
| `$x` | `float` | *not* | The dividend |
| `$y` | `float` | *not* | The divisor |


Return values
----------------

`float`

The floating point remainder of
x/y
