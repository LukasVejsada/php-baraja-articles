Availability in `PHP 4.0`

Finds out whether a variable is a boolean


Parameters
--------------

| Parameter | Data type | Default value | Note |
|-----|-----|-----|-----|
| `$var` | `mixed` | *not* | The variable being evaluated. |


Return values
----------------

`bool`

true if var is a boolean,
false otherwise.
