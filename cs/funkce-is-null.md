Dostupnost ve verzích: `PHP 4.0.4`

Finds whether a variable is &null;


Parametry
--------------

| Parametr | Datový typ | Výchozí hodnota | Poznámka |
|-----|-----|-----|-----|
| `$var` | `mixed` |  | The variable being evaluated. |


Návratové hodnoty
----------------

`bool`

true if var is null, false
otherwise.
