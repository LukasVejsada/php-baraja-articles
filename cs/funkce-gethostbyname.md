Get the IPv4 address corresponding to a given Internet host name


Parametry
--------------

| Parametr | Datový typ | Výchozí hodnota | Poznámka |
|-----|-----|-----|-----|
| `$hostname` | `string` | *není* | The host name. |


Návratové hodnoty
----------------

`string`

the IPv4 address or a string containing the unmodified
hostname on failure.
