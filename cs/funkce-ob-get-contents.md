Dostupnost ve verzích: `PHP 4.0`

Return the contents of the output buffer


Parametry
--------------

Funkce nemá žádné vstupní parametry.

Návratové hodnoty
----------------

`string`

This will return the contents of the output buffer or false, if output
buffering isn't active.
