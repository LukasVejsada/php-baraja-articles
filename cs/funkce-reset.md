Parametry
--------------

| Parametr | Datový typ | Výchozí hodnota | Poznámka |
|-----|-----|-----|-----|
| `$array` | `array` |  | The input array. |


Návratové hodnoty
----------------

`mixed`

the value of the first array element, or false if the array is
empty.
