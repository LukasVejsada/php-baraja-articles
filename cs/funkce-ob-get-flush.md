Dostupnost ve verzích: `PHP 4.3.0`

Flush the output buffer, return it as a string and turn off output buffering


Parametry
--------------

Funkce nemá žádné vstupní parametry.

Návratové hodnoty
----------------

`string`

the output buffer or false if no buffering is active.
