Dostupnost ve verzích: `PHP 4.0`

Finds whether a variable is an object


Parametry
--------------

| Parametr | Datový typ | Výchozí hodnota | Poznámka |
|-----|-----|-----|-----|
| `$var` | `mixed` |  | The variable being evaluated. |


Návratové hodnoty
----------------

`bool`

true if var is an object,
false otherwise.
