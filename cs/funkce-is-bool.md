Finds out whether a variable is a boolean


Parametry
--------------

| Parametr | Datový typ | Výchozí hodnota | Poznámka |
|-----|-----|-----|-----|
| `$var` | `mixed` |  | The variable being evaluated. |


Návratové hodnoty
----------------

`bool`

true if var is a boolean,
false otherwise.
