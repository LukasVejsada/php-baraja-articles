Parametry
--------------

| Parametr | Datový typ | Výchozí hodnota | Poznámka |
|-----|-----|-----|-----|
| `$options` | `string` | *není* |  |

| `$optind` | `int` | null |  |


Návratové hodnoty
----------------

`array`

This function will return an array of option / argument pairs or false on
failure.
