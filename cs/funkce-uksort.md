Parametry
--------------

| Parametr | Datový typ | Výchozí hodnota | Poznámka |
|-----|-----|-----|-----|
| `$array` | `array` |  | The input array. |
| `$cmp_function` | `callback` |  | The callback comparison function. |


Návratové hodnoty
----------------

`bool`

true on success or false on failure.
