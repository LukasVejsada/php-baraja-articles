Finds whether a value is not a number


Parametry
--------------

| Parametr | Datový typ | Výchozí hodnota | Poznámka |
|-----|-----|-----|-----|
| `$val` | `float` |  | The value to check |


Návratové hodnoty
----------------

`bool`

true if val is 'not a number',
else false.
