Finds whether a variable is a resource


Parametry
--------------

| Parametr | Datový typ | Výchozí hodnota | Poznámka |
|-----|-----|-----|-----|
| `$var` | `mixed` |  | The variable being evaluated. |


Návratové hodnoty
----------------

`bool`

true if var is a resource,
false otherwise.
