Dostupnost ve verzích: `PHP 4.0`

Find whether the type of a variable is string


Parametry
--------------

| Parametr | Datový typ | Výchozí hodnota | Poznámka |
|-----|-----|-----|-----|
| `$var` | `mixed` |  | The variable being evaluated. |


Návratové hodnoty
----------------

`bool`

true if var is of type string,
false otherwise.
