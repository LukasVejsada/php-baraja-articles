`print`--字符串输出

描述
--------------------------

```php
print '你好，世界!';
```

`print()`实际上不是一个真正的函数（它是一个语言结构），所以你不需要使用括号。

参数
--------------------------

- **arg**

输出参数

返回值
--------------------------

总是返回数字1。

注意事项
--------------------------

注意：因为这是一个**的语言结构**（不是一个函数），它不能被加载到一个变量中。

例子
--------------------------

```php
print "你好，世界";

print "打印 "可以输出多行文本。但要注意HTML标签的问题。
因为它不会打印。这就是<a href=""/nl2br">Nl2br</a>。";

// 与变量连接的例子
$a = 'ǞǞǞ';

print '我喜欢' . $a; // 我很喜欢php
```

**print**与**echo**的功能完全相同。如果你想了解更多信息，请查看<a href="/echo">echo</a>文章。
