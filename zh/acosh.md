acosh - 双曲余弦

描述
--------------------------

```php
float acos ( float $arg )
```

arg参数的弧度余弦的变化，单位为弧度。acos()是cos()函数的逆函数，这意味着对于acos()函数定义域中的每个值，a==cos(acos(a))。

参数
--------------------------

- **arg**
处理的论据

返回值
--------------------------

参数的弧度，单位是弧度。

与早期版本的不同之处
--------------------------

**5.3.0** - 该功能现在可在所有平台使用
